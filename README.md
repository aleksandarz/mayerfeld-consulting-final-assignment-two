# Mayerfeld Consulting – Final Assignment Project Two

## Live Demo
[Live Demo](https://mayerfeld-consulting-final-assignme-eight.vercel.app/)

## Project Overview
This project is a browser-based jumping game developed through team collaboration. The focus was on building interactive gameplay mechanics, and creating a smooth and responsive experience using vanilla JavaScript.

## Key Features
- interactive jumping mechanics
- score tracking system
- collision detection
- smooth and responsive gameplay
- clean and organized code structure

## Tech Stack
- HTML
- CSS
- JavaScript

## Collaboration
This project was developed as a **group collaboration**, simulating a real development team environment:
- working in shared codebases (Git)
- coordinating tasks and features
- following structured development workflows
- reviewing and improving each other’s code

The experience reflects real-world teamwork and collaboration in developing an interactive browser-based game.

## Purpose
This project demonstrates the team’s ability to:
- collaborate effectively in a shared development environment
- implement core game logic using JavaScript
- handle real-time user input and gameplay interactions
- build an engaging and responsive browser-based game
- maintain clean, structured, and readable code as a team
